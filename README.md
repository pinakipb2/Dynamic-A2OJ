## Development Setup

### Prerequisites

- Install [Yarn] which includes [Yarn Package Manager][yarnpkg]

### Setting Up a Project

Create workspace:

```
mkdir [PROJECT NAME]
cd [PROJECT NAME]
```

Install all packages:

```
yarn install package.json
```

Run the application:

```
yarn start
```
